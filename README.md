
# 1. 项目介绍

该项目是为了学习 VitePress 技术实战，以Pinia官网来练手。 因为 Pinia官网也是有 VitePress 来实现的，所以如果能完美还原1:1, 也是对学习 VitePress 的一种很好的实践。


# 2. 运行项目

## 2.1. 下载项目后，命令行中执行如下命令
```bash
pnpm install
pnpm docs:dev
```

## 2.2. 浏览器中输入  http://localhost:5173/
